# ACS LinkedIn Outreach & Response Framework
**Source:** Wade's GPT planning session (March 5, 2026)
**Status:** Planning Phase — Ready for build
**Owner:** Tré
**North Star:** Walkthroughs booked per week from LinkedIn

---

## SYSTEM PURPOSE
Generate qualified conversations that convert into facility walkthrough appointments.

---

## 1. ICP SEGMENTATION + MESSAGE ANGLES

Segment priority order (fastest decision cycles):

| Priority | Segment | Angle |
|---|---|---|
| 1 | Property Managers / Firms | Reliable vendor support, inspection accountability, responsive comms |
| 2 | Office / Corporate Facility Managers | Consistency, reputation protection, professional standards |
| 3 | Gyms / Fitness Centers | High-touch sanitation, no missed cleaning nights, members notice |
| 4 | Retail / Strip Centers | Simple solutions, reliable recurring service, easy onboarding |
| 5 | Churches | Respectful service, event-ready preparation, consistent care |
| 6 | Financial Institutions / Banks | Trust, professional appearance, background-checked reliability |
| 7 | Schools / Daycares | Safety-focused, structured standards, child-safe environment |
| 8 | Medical / Dental Offices | Compliance awareness, detailed protocols, precision standards |

**Rule:** One segment = one message angle. Never mix angles in outreach.

---

## 2. CONNECTION REQUEST NOTE STRATEGY

Purpose: Gain permission to start a conversation. NOT the sales pitch.

**Formula:** Relevance → Credibility signal → Soft close

**Templates:**

*Property Manager:*
> Hi [Name] — I work with property managers to support reliable, inspection-based cleaning for commercial properties. Would love to connect.

*Facility Manager:*
> Hi [Name] — we help offices maintain consistent cleaning standards with dependable communication and service. Open to connecting.

*Gym Owner:*
> Hi [Name] — we support fitness centers with reliable sanitation and consistent facility cleaning. Would love to connect.

---

## 3. POST-ACCEPT MESSAGE SEQUENCE

### Touch 1 — Day 0–1 (Start the conversation)
> Thanks for connecting, [Name]. We help [facility type] maintain reliable cleaning standards with clear communication and accountability. Quick question — are you currently using an outside cleaning vendor or handling cleaning internally?

### Touch 2 — Day 4–7 (Value + low-friction offer)
> If helpful, I can send our Facility Walkthrough Checklist. It helps identify cleaning gaps and clarify expectations for any facility. If you'd like, I can also do a quick walkthrough and provide a clear cleaning scope summary.

### Touch 3 — Day 10–14 (Respectful close)
> Totally fine if timing isn't right. Would it make sense for me to circle back next month, or is cleaning already handled long-term?

### Optional — Day 30 (Only if prior engagement)
Check-in only if they previously replied or engaged.

**Boundaries:**
- Max 3 messages without response
- No spam follow-ups
- Always professional tone

---

## 4. PERSONALIZATION RULES

**Three Token Rule — use only these three signals:**
1. Role (facility manager, property manager)
2. Facility type (office, gym, clinic)
3. Desired outcome (cleanliness / reliability / sanitation)

**Avoid:** Personal life details, family references, overly specific personal observations

---

## 5. INBOUND RESPONSE PLAYBOOK

**Response time standard:** Within 2 business hours. Next morning if after hours.

**Response structure:** Acknowledge → 2 qualifying questions → offer next steps → confirm scheduling

**Template:**
> Thanks for reaching out, [Name]. To point you in the right direction:
> 1) What type of facility is this?
> 2) Approximately how large is the space?
>
> If you'd like, we can schedule a walkthrough and I can provide a clear cleaning scope and recommendation. Or we can start with a quick 10-minute call.

---

## 6. OBJECTION ROUTING

| Objection | Response |
|---|---|
| "We already have a cleaner" | Most of our current clients already had a cleaning company. They simply wanted a more reliable standard or backup support. If helpful, I could still do a quick walkthrough and provide a scope you can use for comparison. |
| "Not interested" | Understood. Would it be better for me to check back in a few months, or should I close the loop for now? |
| "Send pricing" | Happy to. Pricing usually depends on facility size, frequency, and condition. A quick walkthrough allows us to give an accurate number instead of guessing. |
| "We only work through RFPs" | Perfect. When does the next bid cycle typically open? We can do a walkthrough beforehand so everything is prepared when it does. |

## 7. TRACKING METRICS

**Activity:**
- Connection requests sent
- Connection acceptance rate
- Messages sent
- Replies received

**Positive signals:**
- Any reply
- Request for information
- Scope questions / pricing interest
- Checklist requests

**Conversion:**
- Conversations started → Walkthroughs proposed → Walkthroughs booked → Walkthroughs completed → Quotes sent → Deals closed
